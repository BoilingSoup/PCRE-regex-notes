Description

Desc.

Example

Matches

Does not match

?

Match previous character 1 or 0 times.


(An optional character.)

a?

  • "aaaaa"
  • "bbbbb"
  • " "

Matches with any string

*

Match previous character 0 or more times.


(An optional character. The same character may repeat.)

a*

  • "aaaaa"
  • "bbbbb"
  • " "

Matches with any string

+

Match previous character 1 or more times.

a+

  • "a"
  • "aaaaa"
  • "bbbbb"
  • " "

.

Represent 1 (any) character, but will not match a new line character (\n).

.

  • "a"
  • "9"
  • " "
  • "\n"
  • ""

\

Escape the behavior of a following special character.

\*\*\*

  • "***"
  • "*"
  • "**"
  • "aaa"

^

Indicates the beginning of a line.

^First

  • "First time"
  • "A First"

$

Indicates the end of a line.

last.$

  • "The last."
  • "last week."

{n}

Match the previous character exactly n times.

a{3}

  • "aaa"
  • "a"
  • "aa"

{n,}

Match the previous character at least n times.

a{3,}

  • "aaa"
  • "aaaa"
  • "aaaaa"
  • "a"
  • "aa"

{n, m}

Match the previous character at least n but not more than m times.

a{3, 4}

  • "aaa"
  • "aaaa"
  • "a"
  • "aa"

\w

Match to any alphabet (case insensitive), numeric, or underscore character.

\w

  • "a"
  • "A"
  • "9"
  • "_"
  • " "
  • "%"
  • "\n"

\W

Match to any character that is not alaphabetic, numeric, or an underscore.

\W

  • " "
  • "%"
  • "\n"
  • "a"
  • "A"
  • "9"
  • "_"

\d

Match to any numeric (digit) character.

\d

  • "0"
  • "1"
  • "2
  • "a"
  • "A"
  • "_"

\D

Match to any non-numeric character.

\D

  • "a"
  • "A"
  • "%"
  • "0"
  • "1"
  • "2"

\s

Match to any whitespace character.

\s

  • " "
  • "\n"
  • "1"
  • "a"
  • "%"

\S

Match to any non-whitespace character.

\S

  • "1"
  • "a"
  • "%"
  • " "
  • "\n"

i

Append i to the end of the regular expression to make the regex search case insensitive.

r

  • "r"
  • "R"
  • " "
  • "z"
  • "0"
